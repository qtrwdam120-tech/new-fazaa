# فزعة - Fazaa Membership Website

موقع ويب مطابق لموقع فزعة الرسمي مع خطط الاشتراك والعضوية.

## 📁 هيكل المشروع

```
fazaa-website/
├── index.html          # الصفحة الرئيسية
├── style.css           # ملف الأنماط (CSS)
├── script.js           # ملف JavaScript للتفاعلات
├── README.md           # هذا الملف
└── assets/
    └── images/
        ├── logo.png              # شعار فزعة
        ├── header.jpg            # صورة الهيدر
        ├── products/
        │   ├── Platinum.png      # أيقونة الباقة البلاتينية
        │   ├── Gold.png          # أيقونة الباقة الذهبية
        │   ├── Silver.png        # أيقونة الباقة الفضية
        │   └── Discount.png      # أيقونة خصومات فزعة
        └── slider/
            ├── slider_Fazaa-du-web-banner-arabic.png
            └── slider_Zero-dirhams-arabic.png
```

## ✨ المميزات

### التصميم
- 🎨 تصميم متجاوب (Responsive Design)
- 🌙 الوضع الداكن للفوتر والسلايدر
- 💫 تأثيرات حركية سلسة (Animations)
- 📱 متوافق مع جميع الأجهزة

### الوظائف
- 🔄 Slider تلقائي مع دعم اللمس
- 📋 Accordion لعرض تفاصيل الباقات
- ⬆️ زر العودة للأعلى
- 🔤 زر تبديل اللغة (قيد التطوير)

## 🚀 تشغيل الموقع

### محلياً
```bash
# باستخدام Python
cd fazaa-website
python3 -m http.server 8080

# ثم افتح المتصفح على: http://localhost:8080
```

### بدون خادم
يمكنك فتح ملف `index.html` مباشرة في المتصفح.
